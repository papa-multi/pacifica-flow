# Brand Assets Placeholder

Drop PacificaFlow brand assets here.

Suggested layout:

- `logos/` (`logo.svg`, `wordmark.svg`)
- `icons/` (app + feature icons)
- `fonts/` (brand fonts, if self-hosted)
- `backgrounds/` (hero textures, gradients)

The UI reads from `/public/assets/*` at runtime.
Keep source assets in this folder and publish optimized copies to `public/assets/`.
